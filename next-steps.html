1

Supabaseにテーブルを追加

⏳ 今すぐ実行

プッシュ通知の購読情報を保存するテーブルを作ります。

🔗 https://supabase.com/dashboard/project/mthfqqqukuvueprdokiq/sql
① 上のリンクを開き 「+ New query」 をクリック
② 下のSQLを貼り付けて 「Run」 を実行
③ 「Success. No rows returned」 が表示されたら完了 ✅
-- プッシュ通知購読テーブル CREATE TABLE IF NOT EXISTS push_subscriptions ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, user_id UUID NOT NULL UNIQUE REFERENCES auth.users(id) ON DELETE CASCADE, endpoint TEXT NOT NULL, p256dh TEXT NOT NULL, auth TEXT NOT NULL, created_at TIMESTAMPTZ DEFAULT now(), updated_at TIMESTAMPTZ DEFAULT now() ); ALTER TABLE push_subscriptions ENABLE ROW LEVEL SECURITY; CREATE POLICY "push_sub_insert" ON push_subscriptions FOR INSERT TO authenticated WITH CHECK (auth.uid() = user_id); CREATE POLICY "push_sub_select" ON push_subscriptions FOR SELECT TO authenticated USING (auth.uid() = user_id); CREATE POLICY "push_sub_update" ON push_subscriptions FOR UPDATE TO authenticated USING (auth.uid() = user_id) WITH CHECK (auth.uid() = user_id); CREATE POLICY "push_sub_delete" ON push_subscriptions FOR DELETE TO authenticated USING (auth.uid() = user_id);
2

VAPIDキーを生成する

⏳ 今すぐ実行

「バックグラウンドでプッシュを送る権限証明書」を作ります。ターミナル（Mac: Terminal.app）で1行実行するだけです。

npx web-push generate-vapid-keys
📋 出力された Public Key と Private Key を メモ帳に保存しておく
✏️ index.html の 6452行目あたり の VAPID_PUBLIC_KEY を生成した Public Key に書き換える
⚠️ ターミナルが苦手な場合は、以下のサイトでブラウザから生成できます：
https://www.athenastudio.net/webpush/generate-vapid-keys
3

Edge Function をデプロイ

⏳ 今すぐ実行

フレンドが確定したとき、サーバー側からプッシュを飛ばすプログラムを Supabase にアップします。

① ターミナルでプロジェクトフォルダに移動して以下を実行：
# Supabase CLI をインストール（初回のみ） npm install -g supabase # Supabase にログイン supabase login # プロジェクトをリンク supabase link --project-ref mthfqqqukuvueprdokiq # Edge Function をアップロード supabase functions deploy notify-friend-commit
4

Supabase に環境変数を設定

⏳ 今すぐ実行

VAPIDキーをSupabaseのサーバーに登録します。

🔗 Settings → Edge Functions → Secrets
① 上のリンクを開き 「Add new secret」 を3回クリック
② 以下の3つを入力して保存：
VAPID_PUBLIC_KEY = (Step2で生成したPublic Key) VAPID_PRIVATE_KEY = (Step2で生成したPrivate Key) VAPID_SUBJECT = mailto:your-email@gmail.com
5

Database Webhook を設定

⏳ 今すぐ実行

「daily_statusに確定レコードが追加されたら Edge Function を呼ぶ」トリガーを設定します。

🔗 Database → Webhooks
① 「Create a new hook」をクリック
② 以下の通りに設定して保存：
Name : notify_friend_commit Table : daily_status Events : ✅ INSERT Type : Supabase Edge Functions Edge Function : notify-friend-commit
