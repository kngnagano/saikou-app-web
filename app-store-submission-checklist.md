## 📱 アプリ情報

### 基本情報
- [ ] **アプリ名**: Saikou! （確定）

## 📊 App Store Review 対策

### レビューで確認される項目

#### 1. アプリの完成度
- [ ] すべての機能が正常に動作
- [ ] クラッシュやバグがない
- [ ] ローディング表示が適切
- [ ] エラーメッセージがわかりやすい

#### 2. ユーザーエクスペリエンス
- [ ] 直感的なUI/UX
- [ ] タップエリアが適切なサイズ
- [ ] フォントサイズが読みやすい
- [ ] ボタンの反応が明確

#### 3. コンテンツポリシー
- [ ] 不適切なコンテンツがない
- [ ] 差別的な表現がない
- [ ] 暴力的な表現がない
- [ ] 性的な表現がない

#### 4. プライバシー
- [ ] 収集する情報を明示
- [ ] ユーザーの同意を取得
- [ ] プライバシーポリシーへのリンク
- [ ] 第三者サービスの明記

#### 5. 招待制アプリの対応
- [ ] **デモアカウント情報を提供**
  - 招待コード: `SAIKOU-2026`
  - または既に登録済みのテストアカウント情報
- [ ] レビュアーがすべての機能にアクセス可能

## ⚠️ よくある却下理由と対策

### 1. 機能が不完全
**対策**: すべての機能を実装し、テスト完了後に提出

### 2. クラッシュやバグ
**対策**: 複数デバイスでテスト、エラーハンドリング強化

### 3. プライバシーポリシーが不十分
**対策**: 収集するデータを明確に記載、リンクを正しく設定

### 4. デモアカウント情報が不足
**対策**: レビュアーがすぐに使えるアカウント情報を提供

### 5. UI/UX が不適切
**対策**: Apple Human Interface Guidelines に準拠
