# Saikou! v8.2.0 ゲームエコノミー再設計

## 🎮 設計理念（F2P ゲームのベストプラクティス）

### 目標
1. **ログインだけでは少額** → 習慣化のインセンティブ
2. **挑戦成功で大きな報酬** → コアループの価値向上
3. **デイリー達成でボーナス** → 毎日のモチベーション
4. **経済的に持続可能** → 月間コスト ¥100以下

---

## 📊 新しいポイント獲得システム

### 1️⃣ ログインボーナス（大幅削減）
| 日数 | 旧 | 新 | 理由 |
|------|-----|-----|------|
| 1日目 | 5pt | 1pt | 初回は少額でOK |
| 2日目 | 10pt | 1pt | 継続を促す |
| 3日目 | 15pt | 2pt | 徐々に増加 |
| 4日目 | 15pt | 2pt | - |
| 5日目 | 20pt | 3pt | - |
| 6日目 | 20pt | 3pt | - |
| 7日目 | 25pt | 5pt | 1週間達成ボーナス |
| 8日目以降 | 25pt | 2pt | デフォルト |

**7日間合計**: 110pt → **17pt**（93pt削減）  
**30日間合計**: 750pt → **73pt**（677pt削減）

---

### 2️⃣ デイリーコミットボーナス（新規）
- **3タスク達成**: +3pt（毎日）
- **7日連続達成**: +10pt（ボーナス）
- **30日連続達成**: +30pt（ボーナス）

**30日間の最大獲得**:
- ログイン: 73pt
- デイリー達成: 90pt (3pt × 30日)
- 7日連続ボーナス: 40pt (10pt × 4週)
- **合計**: 203pt

---

### 3️⃣ 挑戦システム（報酬強化）
| レベル | デポジット | 成功報酬 | 純利益 |
|--------|-----------|----------|--------|
| Lv.1 (7日) | 50pt | デポジット返還 + **10pt** | +10pt |
| Lv.2 (10日) | 60pt | デポジット返還 + **15pt** | +15pt |
| Lv.3 (14日) | 80pt | デポジット返還 + **20pt** | +20pt |
| Lv.5 (30日) | 150pt | デポジット返還 + **50pt** | +50pt |
| Lv.10 (90日) | 500pt | デポジット返還 + **150pt** | +150pt |

**変更点**: 成功時にデポジット返還 **＋ ボーナス報酬**

---

### 4️⃣ フレンド招待ボーナス
- **旧**: 50pt
- **新**: 20pt（招待者・被招待者の両方）

**理由**: 招待は一度きりなので、大幅削減しても問題ない

## 📈 ユーザー心理への配慮

### ログインボーナス削減の補填
1. **デイリー達成ボーナス** → 努力が報われる
2. **挑戦成功報酬** → コアループの価値向上
3. **連続達成ボーナス** → 長期継続のインセンティブ

- **成功報酬** → 勝利の喜び（達成感）
