# Saikou! - 招待制システム

## 🔒 招待制の仕様

### **アカウント作成**
- **招待コードが必須**
- 既存ユーザーの招待コード、または管理者コードが必要
- 招待コードが無効な場合は登録不可

---

## 🔑 **管理者用招待コード**

### **初回ユーザー用の特別コード**
```
SAIKOU-2026
```

このコードで最初の4人が登録できます。

**使い方**:
1. 招待コード入力欄に `SAIKOU-2026` と入力
2. 表示名を入力
3. 「始める」をクリック

---

## 👥 **フレンド自動追加機能**

### **招待コードで登録すると**
- 招待した人と自動的に**双方向のフレンド関係**が作成される
- お互いのフレンドリストに即座に表示
- 管理者コードの場合は自動追加なし

---

## 🎯 **4人テストの流れ**

### **1人目（あなた）**
1. 招待コード: `SAIKOU-2026`
2. 表示名: 例「太郎」
3. 登録完了 → 招待コード生成: `太郎-1234`

### **2人目**
1. 招待コード: `太郎-1234` （1人目のコード）
2. 表示名: 例「花子」
3. 登録完了 → 自動的に太郎とフレンドに

### **3人目**
1. 招待コード: `太郎-1234` または `花子-5678`
2. 表示名: 例「次郎」
3. 登録完了 → 招待元とフレンドに

### **4人目**
1. 招待コード: 既存ユーザーのどれか
2. 表示名: 例「美咲」
3. 登録完了 → 招待元とフレンドに

---

## 📋 **招待コードの確認方法**

### **自分の招待コードを見る**
1. プロフィールアイコン（👤）をクリック
2. または、招待ページを開く
3. 招待コードが表示される

### **招待コードをシェア**
1. 招待ページで「コピー」をクリック
2. LINEやメールで送信

---

## 🔧 **技術仕様**

### **招待検証フロー**
```
1. 招待コード入力
   ↓
2. "SAIKOU-2026" ? → YES: 管理者モード
   ↓ NO
3. Supabaseで招待コード検索
   ↓
4. 存在する? → YES: 招待元のuser_idを取得
   ↓ NO: エラー「無効な招待コード」
   ↓
5. 新規ユーザー作成
   ↓
6. フレンド関係を自動作成（双方向）
   - user_id: 新規ユーザー, friend_user_id: 招待元
   - user_id: 招待元, friend_user_id: 新規ユーザー
```

## 🚀 **デプロイ後のテスト**

### **テスト手順**
1. ブラウザのシークレットモードを4つ開く
2. 各ウィンドウで異なるユーザーとして登録
3. 招待コードをコピー&ペースト
4. フレンドリストを確認

## 📝 **注意事項**

### **管理者コード**
- `SAIKOU-2026` は初回テスト用
- 本番環境では変更または削除を推奨
- セキュリティのため、秘密にする

### **招待コードの一意性**
- ユーザー作成時にランダム生成
- 重複の可能性は低いが、本番ではチェック機能を追加推奨
