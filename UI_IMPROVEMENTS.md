## 🔧 技術的な変更

### データフロー
```
1. 「今日を確定」クリック
   ↓
2. saveDailyStatus(true) - Supabaseに保存
   ↓
3. Successバナー表示
   ↓
4. loadUser() - 最新データを取得
   ↓
5. renderHome(currentUser.history) - UIを更新
   ↓
6. 週次ビューにマークが表示される
```

### 重要な関数の変更

#### `handleCommit()`
- Successバナー表示
- ボタン更新
- loadUser()で再読み込み

#### `loadUser()`
- `currentUser.history` を正しい形式で保存
- `done_count` と `is_committed` を含む

#### `renderHome(history)`
- `currentUser.history` を受け取る
- 週次ビューとタスクを更新

#### `renderWeeklyView(history)`
- `history` 配列から各日のステータスを検索
- `is_committed` と `done_count` に基づいてマークを表示

## 📝 今後の改善案（オプション）

### 日付自動検知
現在は手動でページをリロードする必要がありますが、以下で自動化可能：

```javascript
// 日付変更を自動検知
setInterval(() => {
  const lastDate = localStorage.getItem('saikou_last_date');
  const currentDate = getTodayString();
  
  if (lastDate && lastDate !== currentDate) {
    localStorage.setItem('saikou_last_date', currentDate);
    location.reload(); // 自動リロード
  }
}, 60000); // 1分ごとにチェック
```
